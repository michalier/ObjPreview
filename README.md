# OBJ file viewer

My final project for OpenGl class, written in C++.

# Prerequisites

Requires g++ (GNU c++) compiler and Make tool.

# Running

Compile using ``` make ``` command and run.

# Controls

Rotate objects by moving the cursor and holding left mouse button.

Zoom using scroll wheel.

Move the camera by moving the cursor and holding right mouse button.

Close program by pressing the Esc key.
